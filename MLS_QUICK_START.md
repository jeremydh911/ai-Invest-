## âŒ Troubleshooting

### "MLS tab is disabled"
**Problem:** Can't see or access MLS tab
**Solution:** You need to set up credentials first
1. Follow Step 2 above
2. Authenticate successfully
3. MLS tab will become available

### "Authentication failed - invalid credentials"
**Problem:** Can't log in to NWMLS
**Solution:**
1. Verify username and password are correct
2. Try logging in to NWMLS directly at nwmls.com
3. Reset password if needed through NWMLS
4. Re-enter credentials

### "No listings found"
**Problem:** Search returns empty results
**Solution:**
1. Broaden your search criteria
2. Try a different city or county
3. Increase price range
4. Try "all" status instead of just active
5. Check if you're searching in correct area

### "Search is slow"
**Problem:** Search takes too long
**Solution:**
1. Use fewer filters
2. Narrow down geographic area
3. Reduce price range
4. Try searching at off-peak hours
5. Refresh and try again

### "PDF download not working"
**Problem:** Can't download agreement PDF
**Solution:**
1. Check browser pop-up blockers
2. Try a different browser
3. Wait a moment and retry
4. Check internet connection
5. Contact support if persisting
